# 🎓 Smart Attendance Management System

## 📖 Overview  
Backend application built using Spring Boot to manage student attendance through REST APIs.

---

## 🚀 Features  
- Add students  
- Mark attendance  
- View records  

---

## 🛠️ Tech Stack  
Java, Spring Boot, MySQL, Maven  

---

## 📂 Structure  
Controller | Service | Repository | Model  

---

## ⚙️ Run  
1. Create database: `attendance_db`  
2. Configure `application.properties`  
3. Run the application  

---

## 🔗 APIs  
POST /students  
GET /students  
POST /attendance  
GET /attendance
